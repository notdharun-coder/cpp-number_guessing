# 🎯 Number Guessing Game (C++)

A fun console-based number guessing game built using C++.

## 🚀 Features
- Random number generation
- Hint system (Too high / Too low)
- Attempt tracking

## 💻 Tech Used
- C++

## ▶️ How to Run
1. Compile:
   g++ game.cpp -o game
2. Run:
   ./game
